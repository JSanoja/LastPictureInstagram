# LastPictureOn
- Mobile App (Ionic & Angular)
- Show the last picture on Instagram and facebook user.
- APi Instagram
- Api Facebook
- Fontawesome


# Requirements
- Ionic app
- Api Instagram
- Api Facebook

# Install
- git clone https://github.com/JSanoja/LastPictureOn.git
- cd LastPictureOn
- npm install
- ionic serve
